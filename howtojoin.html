How to Join

Joining QMGS MC is easy! Follow these simple steps.

For Java Edition Players: Go to the Multiplayer
section of Minecraft and click "Add Server"
The server name can be anything you like,
but the server address must be:
qmgs-mc.my.pebble.host


Click "Done" and then
select the server from your list to join.

For Bedrock Edition Players: Open Minecraft
and go to the "Play" menu. Click on "Servers"
and then "Add Server". The server name can
be anything you like, and for the server
address, use: qmgs-mc.my.pebble.host

For the port,
use 8071
(the default for Bedrock servers).
Click "Save" and then select the server to join.
